								ИНФОРМАЦИЯ О "GIT"
GIT - программа, которая позволяет работать на локальной машине и взаимодействовать с сервером.
GITHUB - сервер(repozitory) со всеми файлами по проекту.
GIT документация https://git-scm.com/doc 
	git add название документа(не забывать указывать формат документа)(если в названии документа есть пробел, нужно взять все название в кавычки) - позволяет программе добавить документ, и понять с каким документом ведется работа.
	git commit -m название того, как хочешь чтобы был подписан документ(не забывать указывать формат документа)(если в названии документа есть пробел, нужно взять все название в кавычки) - позволяет зафиксировать то, что выполнен какой-то объем работы.
	git status - информация о статусе указанного файла.
	git push - позволяет загружать данные на сервер github.
	git pull - позволяет выгружать измененные данные с сервера github.
	cd - позволяет указать место для перехода на нужный адрес, в папку.
